# 3D World

### Progetto sviluppato per l'esame di Tecnologie Software per il Web
**Università degli Studi di Salerno — A.A. 2023/24**

---

## Obiettivo del Progetto
L'obiettivo principale del progetto è stato lo sviluppo di una web application full-stack basata sull'architettura Client-Server. Il frontend è stato realizzato utilizzando **HTML5**, **CSS3** e **JavaScript** (Vanilla), mentre il backend si interfaccia con il client attraverso l'uso di **Java** e **Servlet**.

## Funzionalità dell'Applicativo
**3D World** è una piattaforma di e-commerce interamente dedicata agli appassionati di stampa 3D, incentrata sulla vendita di un'ampia gamma di filamenti suddivisi per materiali e colorazioni.

La persistenza e la gestione dei dati sono affidate a un database relazionale **PostgreSQL**. L'applicazione offre all'utente le seguenti funzionalità core:
* **Navigazione del catalogo:** Possibilità di esplorare i prodotti disponibili, filtrandoli in base al materiale o effettuando ricerche mirate.
* **Personalizzazione e gestione carrello:** Selezione del filamento, visualizzazione delle varianti di colore disponibili e aggiunta dinamica degli articoli al carrello.
* **Checkout e finalizzazione:** Completamento del processo d'acquisto con contestuale aggiornamento in tempo reale delle quantità in magazzino (stock) e svuotamento del carrello.
